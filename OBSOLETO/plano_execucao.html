Ferramentas
Stack no-code 100% gratuita
⚙️
n8n
Automação e lógica do bot — o cérebro
Recebe mensagens do WhatsApp, processa toda a lógica (qual serviço, qual horário, quem é o cliente), agenda no Google, responde ao cliente. Você monta os fluxos arrastando blocos visualmente.
✓ GRATUITO — self-hosted ou Free Cloud
💬
Evolution API
Conexão do WhatsApp Business
Conecta seu número WhatsApp Business ao n8n via webhook. Open source e self-hosted — sem mensalidade. Alternativa mais simples para começar: Z-API (7 dias grátis).
✓ GRATUITO — open source ⚡ Requer servidor (Oracle Free)
📅
Google Agenda
Calendário central — você já tem
Cada profissional tem seu calendário. O n8n lê a disponibilidade e cria/cancela eventos automaticamente via Google Calendar API. Sincronização em tempo real.
✓ GRATUITO — você já possui
📊
Google Sheets
Banco de dados — você já tem
Armazena clientes, histórico de agendamentos, classificações (regular/VIP), lista de espera e logs. Você visualiza e edita direto na planilha como sempre fez.
✓ GRATUITO — você já possui
🖥️
Glide Apps
Painel do profissional (dashboard)
Transforma sua planilha em um app visual no celular. Você vê agenda do dia, lista de clientes, histórico e métricas. Conecta na Google Sheets sem código.
✓ GRATUITO — até 500 linhas
💳
Mercado Pago
Pagamento de sinal antecipado
Gera links de pagamento Pix via API gratuita. O n8n envia o link ao cliente no WhatsApp automaticamente. Sem mensalidade — paga só quando recebe.
✓ SEM mensalidade ⚡ 3,99% por transação Pix

Cronograma
4 semanas do zero ao lançamento
Semana 1Infraestrutura & Contas
🎯 Meta: todas as ferramentas criadas e conectadas entre si
PASSO 01
Criar conta no n8n
Acesse n8n.io → "Start for free". Será o cérebro de toda automação. Pode começar no cloud gratuito.
⚙️ n8n.io
PASSO 02
Configurar Evolution API
Crie conta grátis no Oracle Cloud (VM free forever) → instale Evolution API via Docker → escaneie QR Code com seu WhatsApp Business.
💬 Evolution API
PASSO 03
Montar planilha de dados
Crie Google Sheets com 5 abas: Clientes, Agendamentos, Serviços, ListaEspera, Logs. Posso te entregar o template completo pronto.
📊 Google Sheets
PASSO 04
Conectar Google Agenda no n8n
No n8n, crie credencial Google Calendar via OAuth 2.0. Autorize acesso à agenda do profissional.
📅 Google Agenda
PASSO 05
Criar conta Mercado Pago
Conta PJ em mercadopago.com.br → área de desenvolvedores → gere o Access Token para a API.
💳 Mercado Pago
Semana 2Bot de Agendamento (Core)
🎯 Meta: cliente consegue agendar pelo WhatsApp do início ao fim
PASSO 06
Boas-vindas + LGPD
n8n recebe mensagem → checa se cliente existe na Sheets → se novo, envia aviso de coleta de dados e aguarda confirmação antes de prosseguir.
⚙️ n8n + Sheets
PASSO 07
Menu de serviços dinâmico
n8n lê aba "Serviços" da planilha e monta automaticamente a mensagem com opções numeradas e preços.
⚙️ n8n + Sheets
PASSO 08
Verificação de disponibilidade
n8n consulta Google Agenda e lista horários livres nos próximos 7 dias. Exibe lista numerada ao cliente para escolha.
📅 Google Agenda
PASSO 09
Confirmação do agendamento
Após cliente escolher: cria evento no Google Agenda, salva na planilha, envia confirmação com resumo completo para cliente e profissional.
⚙️ n8n
PASSO 10
Cancelamento com motivo
Cliente envia "cancelar" → bot pede motivo obrigatório → cancela no Agenda → atualiza planilha → notifica você no WhatsApp.
⚙️ n8n
PASSO 11
Handoff humano
Cliente envia "humano" ou 0 → bot pausa e te notifica com contexto da conversa. Você responde diretamente pelo WhatsApp.
💬 WhatsApp
Semana 3Lembretes, Pagamento & Extras
🎯 Meta: lembretes automáticos funcionando e sinal sendo cobrado
PASSO 12
Lembretes 24h e 2h
No n8n: Schedule Trigger rodando de hora em hora → busca agendamentos próximos → envia lembrete personalizado com localização.
⚙️ n8n Schedule
PASSO 13
Fluxo de atraso
Cliente envia "atraso" → bot te notifica com contexto (quem, horário, serviço) → você responde OK ou NÃO → bot comunica o cliente.
⚙️ n8n
PASSO 14
Cobrança de sinal (Pix)
n8n chama API Mercado Pago para gerar link Pix → envia ao cliente → webhook confirma pagamento → agendamento confirmado automaticamente.
💳 Mercado Pago API
PASSO 15
Reagendamento
Cliente envia "reagendar" → bot verifica limite (máx 2x por RN-09) → exibe próximos 10 slots → cliente escolhe → Agenda atualizado.
⚙️ n8n
PASSO 16
Lista de espera
Horário lotado → bot oferece fila → salva na aba "ListaEspera" → quando há cancelamento, n8n notifica o 1º da fila com 30min para confirmar.
⚙️ n8n + Sheets
Semana 4Painel, Testes & Lançamento
🎯 Meta: painel visual, testes com clientes reais, lançamento
PASSO 17
Painel no Glide Apps
glide.new → conecta na Google Sheets → cria telas: Agenda do dia, Clientes, Histórico. Instala como app no celular. Sem código.
🖥️ Glide Apps
PASSO 18
Testes internos completos
Simule cada fluxo com 2 celulares: agendar, cancelar, reagendar, atraso, lista de espera, pagamento, handoff. Anote e ajuste as mensagens.
🧪 Manual
PASSO 19
Beta com 3-5 clientes reais
Selecione clientes de confiança, avise que está testando o sistema e colete feedback direto. Ajuste o tom das mensagens do bot.
👥 Beta fechado
PASSO 20
🚀 Lançamento oficial
Divulgue o número para todos os clientes. Monitore os fluxos nas primeiras 48h. A plataforma está no ar.
🚀 Go live

Transparência
Limitações honestas do plano gratuito
⚡ Evolution API exige servidor
Precisa instalar em um servidor. Sem servidor = sem WhatsApp funcionando.
✅ Oracle Cloud tem VM gratuita para sempre (Always Free). Posso te guiar na instalação passo a passo.
⚡ Glide: limite de 500 linhas
O plano free suporta até 500 linhas de dados. Suficiente para os primeiros meses.
✅ Softr.io como alternativa gratuita, ou manter visualização direto na Google Sheets até escalar.
⚡ n8n Cloud: 5.000 exec/mês
O plano free do n8n Cloud limita execuções. Volume alto pode estourar o limite.
✅ n8n self-hosted no mesmo Oracle Cloud resolve — sem limite de execuções.
⚡ WhatsApp não-oficial
Evolution API não é a API Meta Oficial. Risco baixo para uso em negócio próprio.
✅ API Meta Oficial é gratuita até 1.000 conversas/mês. Migração simples quando escalar.
⚡ App móvel nativo
O ERS v2.0 especifica app iOS/Android para o profissional. No-code entrega PWA.
✅ Glide gera PWA instalável no celular + notificações via WhatsApp. Funcional para validar o produto.
⚡ Múltiplos profissionais
Gerenciar equipes com agendas separadas é mais trabalhoso no no-code.
✅ MVP com 1 profissional funciona perfeitamente. Expansão para equipe discutimos na próxima etapa.
